「Webコンテンツの作成」の成果

本テーマについて

このテーマでは、HTML,、SS、JavaScriptの基本を学び、これらを組み合わせて Web ページを構成する方法を学ぶ。

学習目標は

  • HTML によって Web ページの構造を記述する基本的な方法がわかる.
  • CSS によって Web ページの見た目を設定する基本的な方法がわかる.
  • JavaScript によって Web ページのインタラクティブな処理を記述する基本的な方法がわかる.
  • HTML, CSS, JavaScript の構文リファレンスから，構文の書式や使用方法の情報を入手できる.
  • HTML, CSS, JavaScript, 画像・音声・動画ファイルを組み合わせて簡単な Web ページが作成できる.

プロジェクト課題について

概要：

  1. グルメ情報についてユーザーが食べ物のジャンルを選択するとWWW のサーバへ検索し、検索結果を表示するページを作成する。
  2. 検索内容に応じてお店の情報が出てくる。
  3. 1回目から6回目までの学習内容を使いページをレイアウト、表示させる。
    4. プロジェクト課題で作成したページ:project.html

    数当てゲーム

    概要：1回目

    1. 乱数で1〜10の値を1つ決めて、「答え」とし、予想は”4”で固定する。
    2. 自動的に3回まで予想する。
    3. 予想の4と「答え」があっているなら「正解です。おめでとう！」と表示し、2回目以降は「答えは ◯ でした．すでにゲームは終わっています」と表示する。

    概要：2回目

    概要：3回目

    1. 乱数で1〜10の値を1つ決めて、「答え」とし、ユーザは答えを予想して、数を最大で3回入力できる。
    2. ユーザの入力が外れた場合は、入力よりも答えが大きいか小さいか表示する。
    3. 3回以内に答えの数を当てればユーザの勝ち、そうでなければ負け。
    4. ページにはテキストボックスとボタンが設置されていて、ユーザはテキストボックスに予想した数を入力し、ボタンを押すと判定される。

    数当てゲームのページ:kazuate.html

各授業回の学習事項

第1回 WWWとHTML

第1回目はHTML について学んだ。HTMLとはWeb ページの構造を記述するための書式で、 HTML文書を作ることによってWebページに掲載するコンテンツと、ページの基本的なレイアウトを設定できる。 今回のHTMLの要素は画像の埋め込み方とハイパーリンクの設置の仕方、色々なテキストの要素について学んだ。

演習の成果:演習1-4:HTMLのさまざまな要素

第2回 CSSによるWebページの装飾

第2回目はCSSについて学んだ。CSSとはページの装飾をするものでHTMLの後に書く編集専用ファイルのようなもの。 装飾は数多くあるのでMmdn Web docsで探すと便利。Fire boxでweb開発ツールが使えてそのページのHTMLとCSSの設定状況が見れる。

演習の成果:演習2-3:ウェブブラウザ説明ページの装飾

第3回 JavaScriptの基本

第3回目はJavaScript について学んだ。JavaScriptはWebページ上で複雑な機能を実現するプログラミング言語で、 Web ページを開いた直後や、ボタンを押したときにJavaScriptプログラムが実行される。 ページの自動更新やアニメーションなど動きのあるページを実現するためにJavaScriptプログラムを使う。

> 演習の成果:演習3-6:オブジェクトの参照

第4回 DOMとJavaScript

第4回目ではDOMについて学んだ。DOMとはWebコンテンツを表現するデータ構造のこと。 DOMを使用することによってJavaScript などのプログラムによって、Web コンテンツの内容を読み取ったり、 コンテンツに変更を加えたりする処理が書けるようになる。やり方としてはFirefoxでhtmlを開いてから ウェブ開発ツールのコンソールからdocument.querySelectorで書き加えたい文の箇所を探し、 document.createElementなどで文を追加していったりする。できたらコードをjsに追加する。

演習の成果:演習4-5:オブジェクトの参照

第5回 イベント処理

第5回ではイベント処理について学んだ。イベントは検索ボタンなどを押したりエラーが発生したときなどのWebページ上で起こった動作や出来事のこと。 イベント処理の仕方はイベントハンドラの登録（何が発生したら）とイベントハンドラの定義（イベントが発生した時どういう処理をするか）か必要である。

演習の成果:演習5-3:オブジェクトの参照
